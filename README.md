# Shorts Auto Next

YouTube Shorts에서 자동으로 다음 쇼츠로 넘기는 Chrome 확장프로그램 (Manifest V3)

**버전**: 2.0.0

## 기능

- ✅ YouTube Shorts 재생이 끝나면 자동으로 다음 쇼츠로 이동
- ✅ `Alt+N` 키로 ON/OFF 토글
- ✅ 상태는 브라우저 재시작 후에도 유지
- ✅ 사용자 수동 이동 감지 (수동으로 넘기면 자동 이동 일시 중지)
- ✅ 중복 실행 방지 (2개씩 넘어가는 문제 해결)
- ✅ 쇼츠 종료 후 0.3초 딜레이 후 이동 (자연스러운 전환)
- ✅ 안정적인 비디오 감지 및 바인딩

## 설치 방법

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 폴더(`shorts-auto-next`) 선택
5. 설치 완료!

## 사용 방법

- 확장프로그램은 기본적으로 **ON** 상태로 시작됩니다
- `Alt+N` 키를 눌러 ON/OFF를 토글할 수 있습니다
- 토글 시 화면 우측 하단에 상태가 표시됩니다
- 쇼츠가 끝나면 자동으로 0.3초 후 다음 쇼츠로 이동합니다
- 수동으로 `↓` 또는 `↑` 키를 눌러 넘기면, 이후 2초 동안 자동 이동이 일시 중지됩니다

## 주요 개선 사항 (v2.0.0)

- 🐛 **버그 수정**: 쇼츠가 2개씩 넘어가는 문제 해결
- 🐛 **버그 수정**: 수동 이동 시 자동 이동이 한 번 더 실행되는 문제 해결
- ✨ **새 기능**: 사용자 수동 이동 감지 및 자동 이동 일시 중지
- ⚡ **개선**: 쇼츠 종료 후 딜레이 추가 (0.3초)로 더 자연스러운 전환
- 🔧 **개선**: 중복 실행 방지 로직 강화
- 🔧 **개선**: 오류 처리 및 안정성 향상
